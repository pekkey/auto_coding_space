## 📦 安装命令

```bash
# 安装单个技能
clawhub install <skill-name>

# 例如：
clawhub install tavily-search
clawhub install self-improving-agent
clawhub install code
clawhub install pdf-extract

# 批量安装推荐技能
clawhub install tavily-search code data-analyst automation-workflows pdf-extract self-improving-agent agent-team-orchestration

# 查看已安装的技能
clawhub list

# 更新已安装的技能
clawhub update --all
```
